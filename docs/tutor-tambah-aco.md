# Tutorial Menambahkan Gambar Menu

tutor tambah gambar menu pada website Warkop Azzahra.

---

## Persiapan

Sebelum dimulai bosku, pastikan punyaki :
- Foto menu dalam format `.jpg` atau `.png`
- Akses ke folder project

---

## Langkah 1: Buka Folder Gambar

Buka folder berikut di project:

```
public/assets/menu/
```

Di dalam folder tersebut ada 3 subfolder:
- `beverages/` - untuk gambar minuman
- `foods/` - untuk gambar makanan
- `promotions/` - untuk gambar promosi

---

## Langkah 2: Ubah Nama File Gambar

Ubah nama file foto sesuai dengan daftar nama file di bawah.

Contoh:
- Foto kopi hitam → rename menjadi `kopi-hitam.jpg`
- Foto nasi campur → rename menjadi `nasi-campur.jpg`

---

## Langkah 3: Pindahkan Gambar ke Folder

Pindahkan file gambar ke folder yang sesuai:
- Gambar minuman → `public/assets/menu/beverages/`
- Gambar makanan → `public/assets/menu/foods/`
- Gambar promosi → `public/assets/menu/promotions/`

---

## Langkah 4: Refresh Browser

Buka browser dan akses `localhost:3000`, lalu refresh halaman. Gambar yang baru ditambahkan akan langsung tampil.
